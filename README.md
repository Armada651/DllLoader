**What is this?**

It's a dll that loads other dlls into Fallout 4 via X3DAudio1_7.dll.

**Compile Notes**

- HJWasm must be setup before opening the project.

**NOTES**

- Written in C++/ASM (Visual Studio 2015 & HJWasm).
